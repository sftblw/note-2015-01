좀 있다 코드를 3개 짬
---------------------

-	프로그램을 여러개 합쳐서 컴파일 할 때 makefile 사용
-	.o 파일을 합친다
-	20개의 파일을 하나로 합칠 때... 20번 컴파일하는 대신 make 사용

```Makefile
# Hi! this is a simple makefile!

test:main.o func.o func2.o
	gcc -o test main.o func.o func2.o

main.o:main.c
	gcc -c main.c

func.o:func.c
	gcc -c func.c

func2.o:func2.c
  gcc -c func2.c
```
